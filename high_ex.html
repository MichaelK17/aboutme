  • Close up image of the gadget. It displays a picture book of "Cinderella"
    iPad is placed below, so that the light reflects on an acrylic board. When seen from a certain degree, it looks like it's floating in the air.
    A description of the gadget. Titled as "Luminous picture book"
    This "Luminous picture book" gadget works by using an illusion technique called "Pepper's ghost" (click here). It works by placing a light source, which is projected on an angled transparent board, which is then shown to the audience.

    When the audience sees the light source from a certain angle, it looks like the object projected by the light source is floating. This technique is famously used in Haunted Mansion and Tower of Terror in Disney World.
